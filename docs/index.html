网易云音乐缓存在线解码

选择本地缓存文件

默认缓存目录

  • Windows 默认为 C:\Users\你的用户名\AppData\Local\Netease\CloudMusic\Cache\Cache
  • MacOS 默认为 /Users/你的用户名/Library/Containers/com.netease.163music/Data/Caches/online_play_cache
  • Linux 默认为 NEED_HELP
